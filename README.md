# Superstore Sales Analysis

## Objective
Analyze Superstore sales data using data storytelling, visualization, and statistical validation to identify business insights.

## Dataset
Superstore Dataset (CSV)

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- SciPy

## Analysis Performed
- Sales by Category
- Profit by Category
- Sales by Region
- Top 10 States by Sales
- Sales by Customer Segment
- Sales by Ship Mode
- Correlation Heatmap
- Hypothesis Testing

## Outcome
Generated business insights and recommendations based on sales and profit performance.
